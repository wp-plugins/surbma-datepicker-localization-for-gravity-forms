Localize the Datepicker field with this plugin's automatic localization.

== Description ==

Localize the [Gravity Forms](http://surbma.hu/go/gravityforms/) (affiliate link) Datepicker field with this plugin's automatic localization. There is no need to do anything, just activate the plugin. The js file will load only with the [Gravity Forms](http://surbma.hu/go/gravityforms/) (affiliate link) form, so there will be no unwanted extra load for your website.

== Installation ==

1. Upload `surbma-datepicker-localization` folder to the `/wp-content/plugins/` directory
2. Activate the Surbma - Datepicker localization for Gravity Forms plugin through the 'Plugins' menu in WordPress
3. That's it. :)
